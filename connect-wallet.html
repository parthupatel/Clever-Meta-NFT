Meta Mask

MetaMask is a software cryptocurrency wallet used to interact with the Ethereum blockchain. It allows users to access their Ethereum wallet through a browser extension or mobile app

Bitski

Bitski is a custodial wallet where everything is encrypted so that you can trust that nobody else can view your passcodes. By hosting everything in one place, Bitski also allows users to easily migrate between devices.

Fortmatic

Fortmatic is a User-Friendly Ethereum DeFi Wallet. It is an alternative to a web3 wallet like Metamask. It allows users to use a phone number or email address to connect to Ethereum-based applications.

Wallet Connect

WalletConnect is an open-source protocol that helps implement established connections between various crypto wallets and decentralized finance (DeFi) DApps.

Coinbase Wallet

This means that the private keys (that represent ownership of the cryptocurrency) for your Wallet are stored directly on your mobile device and not with a centralized exchange like Coinbase.com.

Authereum

Authereum is an Ethereum-based wallet and dApp login solution. It allows users to easily interact with web 3 applications via any browser, using nothing but a username and password.

kaikas

Kaikas is a global digital wallet functioning as a web browser extension. With Kaikas, you can safely and seamlessly manage the Klaytn-based tokens that you control.

Torus

It is the mainstream gateway for users to access decentralized ecosystems via their personal and social media accounts such as Google, Facebook, Discord and more.

Bitcoin

Bitcoin wallets contain a user's keys, not their actual bitcoin. Conceptually, a wallet is like a keychain in the sense that it holds many pairs of private and public keys. These keys are used to sign transactions
